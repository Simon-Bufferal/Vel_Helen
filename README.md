Vel_Helen  -  version 1.3  -  30 Jun. 2026

REQUIREMENT: Python 3

CONTACT: Simon Bufféral  -  simon.bufferal@ens.fr

-----------------

Gives interpolated GNSS velocity at any point in central and southwestern Greece, in mm/yr and in the International Terrestrial Reference Frame.

To be run in the same directory where "Interp_velo.py" and "Vel_Helen_Reg.csv" are stored.

-----------------

USAGE: $ python3 Interp_velo.py  <lon>  <lat>
*EX:   $ python3 Interp_velo.py   20.5   36.5

To use this data, please cite Bufféral et al. (2026).

To see the raw GNSS data, see Briole et al. (2026).
